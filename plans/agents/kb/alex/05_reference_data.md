## Common questions with canonical answers

**Q: "What AI do you use?"**
A: "Atlas — AMG's proprietary engine. It's the platform Solon built to run every AMG service."

**Q: "How's this different from just using ChatGPT?"**
A: "ChatGPT is a tool; this is an operator. Seven agents trained on your business specifically, running 24/7, reviewing each other's output, handing work between each other. You don't run a prompt — you have a team."

**Q: "How long to go live?"**
A: "Member portal: ~2 weeks. Chamber OS core modules: Month 1. Extended modules: rolled out over 90 days per Board direction."

**Q: "What's the minimum member commitment?"**
A: "None for Chambers. Members pay monthly, cancel anytime. You earn rev-share on every active member — break-even for the Chamber is member #1."

**Q: "What about our existing members' websites?"**
A: "Additive, not a migration. Members keep whatever they have. AMG sits alongside + adds the 7 agents and dashboard."

**Q: "Price for the Chamber itself?"**
A: "For Revere specifically: zero, or 50% off everything if the Board wants to subscribe Revere itself to Chamber OS Pro as professional-courtesy."

**Q: "How do I know Chamber rev-share is being calculated correctly?"**
A: "Monthly rev-share statement auto-generated + delivered to the Chamber admin dashboard. Every member subscription that attributes to your Chamber is line-itemed."
